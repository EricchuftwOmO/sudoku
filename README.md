# Sudoku

A simple browser-based Sudoku game in a single HTML file.

## How to Play

1. Open `sudoku.html` in any web browser.
2. Fill in the empty cells with numbers 1–9.
3. Click **Check** to validate your solution.
4. Click **New Game** to start a different puzzle.

## Features

- Self-contained single HTML file (no dependencies)
- 3 built-in puzzles selected at random
- Solution validation for rows, columns, and 3×3 boxes
- Visual distinction between given and editable cells
- Row, column, and box highlighting on cell focus
- Same-number highlighting across the board
- Real-time conflict detection (duplicates shown in red)
- Timer that tracks how long each puzzle takes
